LOCKSS Software Releases

LOCKSS Release Notes

Released Daemons

Daemon 1.3.6 (released 8/11/2004)

  • addition of the audit proxy port
  • plug-ins are now downloadable separately
  • new xml prop files are supported
  • UI comes up with warning until all AUs are started
  • tiny UI comes up when daemon is unable to reach prop server
  • initial cut of alerts framework

Future Releases

Daemon 1.4 (freeze early September, release late September)

  • fixes for AU scaling problems seen around 60 AUs
  • further support for downloadable plug-ins
  • URL canonicalization (for IOP and HighWire)
  • define AU by issue ranges
  • refactoring of plug-in code
  • LockssApp pulled out of LockssDaemon
  • minor improvements to UI for adding many AUs

Daemon 1.5 (freeze mid October, release early November)

Meant to be a stability release, we're trying to keep new features to a minimum (except perhaps for UI fixes)
  • refinement of the Alerts framework
  • UI for configuration of Alerts
  • further UI improvements for adding many AUs
  • improving code test coverage
  • general bug fixes
  • ground work for the new protocol

Deamon 1.6 (freeze late November release mid January 2005)

Feature release, hopefully with some portion of the new protocol.
  • OAI crawling
  • initial version of the new protocol (may not be complete)
  • "Cache everything I have access to" feature in UI
